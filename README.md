Запуск:

Хост: [http://localhost:9000/](http://localhost:9000/)

Время: 6-7 часов


#### Сложности и что бы улучшил:
Быстро не удалось придумать хорошего способа отрисовки UI, поэтому за отрисовку UI и запуск игры отвечает один файл `controller.ts`

Код игры сейчас принимает на вход букву введенную пользователем, сам меняет своё внутреннее состояние и запускает событие onUpdate когда состояние игры обновилось. Получилось не очень гибко, возможно стоило управлять состоянием тренировки самостоятельно, снаружи, через API.
